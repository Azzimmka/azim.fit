# KEEP AT IT — автоматическое закрытие уведомлений

## Поведение

- Обычные уведомления автоматически закрываются через 3 секунды.
- Уведомление удаления с действием «Отменить» остаётся доступным 8 секунд.
- Кнопка закрытия продолжает работать мгновенно.
- Наведение, фокус и тип уведомления не приостанавливают отсчёт.

## Реализация

Общий компонент `Toast` получает длительность показа и запускает один очищаемый таймер. По умолчанию используются 3 секунды. Уведомление удаления передаёт 8 секунд явно, поэтому существующее окно отмены и срок доменного snapshot остаются синхронизированы.

При замене уведомления таймер перезапускается. При ручном закрытии и размонтировании компонента таймер очищается, чтобы старое уведомление не могло закрыть новое.

## Проверки

- Обычное уведомление вызывает `onDismiss` через 3 секунды.
- Уведомление с отменой использует 8 секунд.
- Ручное закрытие остаётся доступным.
- Замена или размонтирование уведомления очищает старый таймер.
